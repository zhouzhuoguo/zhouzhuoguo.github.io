Current Research

üíª Chip performance evaluation modeling

Simulation of mainstream AI models(Diffusion, Transformer, CNNs) on customized chip ¬∑ modeling (PPTX)

  • Accurate:
    • Behaves like hardware on each clock rising edge
    • Instantaneous EMA, utilization, power, etc.
  • Complete:
    • Covers all modules: interfaces, NoC, memory management (allocate, release, conflict)
    • Models real dataflow: on-chip, off-chip, and inter-module communication
  • Scalable: Speed vs Granularity
    • Cycle-accurate (1 cycle/step): ~ 6 hours
    • Stride mode (100 cycles/step): ~10 minutes (about 15% deviation)

üíº Latest progress

    • Decoupled: Separated the model into the main engine with editable HW_Setting and ARCH for easy swap of network models.
    • Runtime: Diffusion example reduced from 50h+ to <10min.
    • Dataflow: Supports cross-layer routing and multi-source inputs.
    • Process-aware: Configurable manufacturing process; reports runtime, energy, power, off-chip energy, and access counts.
    • Bit-width: Data bit width is configurable.
    • Local attention: Aggregates per-stage metrics and scales for large repeated stages.
